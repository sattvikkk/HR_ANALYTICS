# HR_ANALYTICS
HR Analytics Employee Attrition &amp; Performance Analysis

Author: Satvik Tiwari


## Summary
This end-to-end HR Analytics project analyzes employee records to identify the key drivers of workforce attrition. The project spans the full data pipeline — from raw CSV ingestion and SQL-based KPI validation to an interactive Power BI dashboard and a professionally formatted analytical report.
